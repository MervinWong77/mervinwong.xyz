# CopyCat product screenshots

Single source of truth for `/download` and `/download/copycat`.

Replace these files in place when the macOS UI changes — do not edit page code
to point at new paths.

| File | Screen |
|------|--------|
| `home.png` | Home (locations selected) |
| `scanning.png` | Scanning in progress |
| `review.png` | Duplicate review |
| `cleanup.png` | Cleanup confirmation |
| `empty.png` | Empty results |

Export via `COPYCAT_SNAPSHOT=visual-system` in the CopyCat macOS app, then copy
the matching acceptance PNGs into this folder with the names above.
