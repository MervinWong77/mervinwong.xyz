# CopyCat downloads

Static hosting for CopyCat release builds and assets.

```
downloads/copycat/
├── brand/        # App icon + brand mark
├── mascot/       # Mascot illustrations for feature sections
├── latest/       # Current private-beta build (CopyCat.dmg)
└── releases/     # Versioned archives (optional)
```

Product UI screenshots for `/download` live in:

`public/projects/copycat/screenshots/`

Served at `/downloads/copycat/...` via the Next.js `public/` directory.
