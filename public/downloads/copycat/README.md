# CopyCat downloads

Static hosting for CopyCat release builds and assets.

```
downloads/copycat/
├── brand/        # App icon used by /download pages
├── mascot/       # Reserved for future marketing use
├── screenshots/  # Reserved for future marketing use
├── latest/       # Current private-beta build (CopyCat.dmg)
└── releases/     # Versioned archives (optional)
```

Served at `/downloads/copycat/...` via the Next.js `public/` directory.

Current private beta artifact:

- `latest/CopyCat.dmg`
