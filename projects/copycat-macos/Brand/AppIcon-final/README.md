## Composition (locked structure)

- Treatment: **05 – Inverted Plate**
- Plate: solid accent colour (or charcoal for grey study)
- Mark: near-white `#F5F5F5`
- Corner radius: ~23.5% (softer macOS feel)
- Mark scale: **0.728** (~+4% vs prior 0.70)
- Optical offset: (−6, +10) on 1024 canvas
- Depth: optional under-layer at ~18% opacity, +3px — not a visible shadow
