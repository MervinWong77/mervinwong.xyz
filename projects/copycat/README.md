# CopyCat — design & product assets

Source of truth for CopyCat branding, design deliverables, documentation, and screenshots.

App source code lives in the sibling `copycat-macos` repository.

```
projects/copycat/
├── assets/
│   ├── mascot/     # Character storyboards and mascot art
│   ├── icons/      # App icon masters, explorations, finals
│   ├── effects/    # Visual effects (reserved)
│   └── logo/       # Brand marks
├── branding/       # Icon guidelines and branding explorations
├── design/         # Design handoff, product design, UI rounds
├── docs/           # Architecture and engineering notes
└── screenshots/    # Product screenshots
```

Downloadable builds will be published under `public/downloads/copycat/` and surfaced at `/download/copycat`.
